----------------------------------------------------------------------------------------------------------------
SUMMARY
This face detection application is built using Python and OpenCV. It utilizes the Haar Cascade Classifier for detecting faces in real-time through a webcam feed.

----------------------------------------------------------------------------------------------------------------
INSTALLATION
- Ensure the following components are installed before running the script::

    1. Python: The script is developed in Python (3.12.3).

    2. OpenCV:         
        Install 1: pip install opencv-python 
            - For processing images and video streams 
        Install 2: pip install opencv-python pillow
            - For image handling in Tkinter

----------------------------------------------------------------------------------------------------------------
RUNNING THE APPLICATION
    - To start the application, run the `main.py` script using Python:
        >> python gui.py

EXITING THE APPLICATION
    - To exit the application, simply close the GUI window or use the "Stop Camera" button provided in the GUI.

----------------------------------------------------------------------------------------------------------------
PURPOSE
This script demonstrates:
    - The use of the Haar Cascade Classifier for effective real-time face detection.
    - Practical application of OpenCV for image and video processing in Python.
    - Techniques for integrating real-time video processing with machine learning models in a user-friendly interface.
